## Gameplay

1. **Produce**: Start by keeping an eye on your basic generators.
2. **Upgrade**: Use your resources to make generators faster.
3. **Fuse & Split**: Go to the Fusion tab to make new elements, or use Fission if you're low on energy!
4. **Grow**: Keep scaling up until you've unlocked everything!
